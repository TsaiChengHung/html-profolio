# HTML Portfolio

A personal portfolio website showcasing web development projects.

## Overview

This is ChengHung Tsai's portfolio website built with pure HTML and CSS. It features a clean, simple design that highlights various web development projects and provides contact information.

## Projects Included

- **Movie Ranking Project** - A curated list of favorite movies
- **Birthday Invite Project** - An interactive birthday invitation page

## Features

- Clean and minimalist design
- Responsive layout
- Easy navigation
- Project previews with screenshots
- About Me and Contact pages

## Structure

```
html-profolio/
├── index.html           # Main portfolio page
├── solution.html        # Alternative solution
├── goal.png            # Design goal reference
├── assets/
│   └── images/         # Project screenshots
└── public/
    ├── about.html      # About page
    ├── contact.html    # Contact page
    ├── movie-ranking.html
    └── birthday-invite.html
```

## Getting Started

Simply open `index.html` in your web browser to view the portfolio.

```bash
# Clone the repository
git clone https://github.com/TsaiChengHung/html-profolio.git

# Navigate to the directory
cd html-profolio

# Open in browser
open index.html
```
